# 2026-ArtsTas-SL
Grant application for Arts Tasmania — Secret Lab — Organisation funding for new projects emphasising Hobart-based/Arts focus.

## Application Documents

- [Full Grant Application](GRANT_APPLICATION.md) - Comprehensive application for Arts Tasmania Organisation Funding

## Overview

This repository contains Secret Lab's 2026 grant application to Arts Tasmania for organisation funding. The application emphasises:

- **Hobart-based operations and focus**
- **Arts-centric programming and outcomes**
- **Community engagement through creative practice**
- **Support for local artists and creative practitioners**

## Key Features

The proposed project includes:
- Artist development programs and residencies
- Public arts events and exhibitions
- Community workshops and engagement activities
- Professional development for Hobart-based artists
- Documentation and promotion of local arts scene

## Application Details

- **Funding Category:** Organisation Funding for New Projects
- **Project Duration:** 18 months
- **Primary Location:** Hobart, Tasmania
- **Focus Areas:** Artistic excellence, community engagement, local artist support

For complete application details, see [GRANT_APPLICATION.md](GRANT_APPLICATION.md).
